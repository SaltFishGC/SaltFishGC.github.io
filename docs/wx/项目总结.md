### LetsChat —— 即时通讯与音视频通话平台
技术栈： SpringBoot MyBatisPlus MySQL Redis Redisson Netty Electron Vue3 Sqlite

项目描述： 设计并实现了一个支持单聊、群聊、文件传输、音视频通话的即时通讯系统，采用 Netty WebSocket 实现长连接通信，支持分布式集群部署，前端基于 Electron 构建跨平台桌面应用。

- **高性能WebSocket服务**：基于Netty框架实现WebSocket长连接服务，支持心跳检测，通过**WebSocket**实现**在线消息推送**/**WebRTC**等服务器到客户器连接
- **WebRTC实时音视频通话**：完整实现**WebRTC信令交换机制**，支持发起通话、接听/拒绝/忙线响应、挂断等全流程，使用 Redis Hash 存储通话房间信息，维护用户通话状态
- **智能离线消息策略**：采用**读写扩散**实现**离线消息推送**，用户单播**写扩散**推送至Redis收件箱，群组根据人数**混合使用读写扩散**，提升查询效率
- **Redis 多级缓存体系**：构建全面的缓存架构，包括用户心跳检测、Token认证管理、联系人列表缓存、节点连接信息维护、WebRTC 房间状态管理等，使用Redis Set/ZSet实现离线消息收件箱
- **分布式集群架构**：使用**Redisson**实现集群节点间消息广播，通过Redis维护节点信息与用户连接映射关系，支持水平扩展
- **鉴权与权限管理**：使用**拦截器**+**AOP**实现统一的权限验证和登录状态检查，支持管理员权限校验，通过**ThreadLocal**传递用户上下文信息
- **OSS对象存储**：使用阿里云OSS存储用户上传的文件，文件大小限制等设置可配置，支持文件下载权限校验
- **MQ消息队列**：使用**RabbitMQ**优化消息聊天消息的数据库写入操作，实现**异步**处理，提升系统吞吐量
- **Electron 跨平台桌面应用**：基于Electron + Vue3 + Pinia构建现代化桌面应用，集成**SQLite**本地存储，实现消息本地缓存、文件管理、用户设置等完整客户端功能
