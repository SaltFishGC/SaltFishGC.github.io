### 四种for循环

| 特性           | `for` (传统)            | `for...in`     | `for...of`      | `forEach`                |
| -------------- | ----------------------- | -------------- | --------------- | ------------------------ |
| **适用对象**   | 数组、类数组            | 对象（属性）   | 可迭代对象      | 数组、类数组             |
| **遍历内容**   | 索引                    | 属性名（键）   | 元素值          | 元素值 + 索引            |
| **是否可中断** | ✅ (`break`)             | ✅ (`break`)    | ✅ (`break`)     | ❌（`return` 仅跳过当前） |
| **是否可跳过** | ✅ (`continue`)          | ✅ (`continue`) | ✅ (`continue`)  | ❌（`return` 仅跳过当前） |
| **获取索引**   | 直接 (`i`)              | 需 `key`       | 需 `.entries()` | 回调参数 `index`         |
| **获取值**     | `arr[i]`                | `obj[key]`     | 直接            | 回调参数 `item`          |
| **遍历原型链** | 否                      | 是（需注意）   | 否              | 否                       |
| **遍历顺序**   | 明确                    | 不保证         | 插入顺序        | 按索引顺序               |
| **性能**       | ⭐⭐⭐⭐⭐ (最快)            | ⭐⭐             | ⭐⭐⭐⭐            | ⭐⭐⭐                      |
| **函数式风格** | 否                      | 否             | 否              | ✅                        |
| **异步支持**   | ✅（配合 `async/await`） | ✅              | ✅               | ❌（不等待异步）          |
| **ES 版本**    | ES3                     | ES5            | ES6             | ES5                      |

| 场景                              | 推荐方式                                                     |
| --------------------------------- | ------------------------------------------------------------ |
| 遍历数组，需要高性能或精确控制    | ✅ 传统 `for`                                                 |
| 遍历数组，代码简洁，无需中断      | ✅ `forEach`                                                  |
| 遍历数组，需要 `break`/`continue` | ✅ `for...of`                                                 |
| 遍历字符串、Set、Map 等可迭代对象 | ✅ `for...of`                                                 |
| 遍历普通对象的自身可枚举属性      | ✅ `for...in` + `hasOwnProperty` 或 `Object.keys()` + `for...of` |
| 函数式编程风格，链式调用          | ✅ `forEach`（或 `map`、`filter` 等）                         |
