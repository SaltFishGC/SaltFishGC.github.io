### WebRTC实现实时通话

#### 新建窗口

ipc中准备了一个newWinfow，只需要在renderer的router里面准备好path和vue就可以实现eletron新开一个页面了：

```javascript
// 新窗口打开
const onOpenNewWindow = () => {
	ipcMain.on("newWindow", async (e, config) => {
		openWindow(config);
	});
};
```
