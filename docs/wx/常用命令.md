### Redis 启动命令

```cmd
redis-server.exe
```

（配置环境变量后）

### 运行后端jar包

```cmd
java -Dws.port=8081 -Dserver.port=8080 -jar your-application.jar
```

（JVM参数）

或者

```cmd
java -jar your-app.jar --server.port=8090 --ws.port=8091
```

（命令行参数）

两种选一个，并记得加上包名称

注意java版本最低**jdk11**（用了mybatisplus的分页组件）

### 运行前端项目（测试环境）

```
npm run dev
```

用node做的npm都是这一套
