### Redis 启动命令

```cmd
redis-server.exe
```

（配置环境变量后）

### 运行jar包

```cmd
java -jar 
```

加上包名称

注意java版本最低**jdk11**（用了mybatisplus的分页组件）

### 运行前端项目

```
npm run dev
```

用node做的npm都是这一套
