date: 2026-2-20

做小项目的时候我们一般不考虑对request信息进行加密，但是实际生产环境即便有ssl作安全保证也必须要进行加密。

| 保护层面         | SSL/TLS          | 应用层加密(RSA/AES) |
| :--------------- | :--------------- | :------------------ |
| **传输中数据**   | ✅ 保护           | ✅ 可额外保护        |
| **存储数据**     | ❌ 不保护         | ✅ 可保护            |
| **日志数据**     | ❌ 不保护         | ✅ 可保护            |
| **数据库泄露**   | ❌ 无法防御       | ✅ 可防御            |
| **内部人员访问** | ❌ 无法防御       | ✅ 可防御            |
| **端到端加密**   | ❌ 服务器可见明文 | ✅ 服务器也不可见    |

也即存到数据库也可以加密

> 当然加密解密也需要额外资源，带来的性能损耗必须在可接受范围之内。
