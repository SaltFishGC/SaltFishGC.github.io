### 装机Man —— 硬件评测与智能装机推荐平台

**技术栈：** SpringBoot MyBatisPlus MySQL Redis    JWT Nginx Vue

**项目描述：** 设计并实现了一个面向 PC 硬件爱好者的点评与装机推荐系统，支持硬件商品浏览、用户评测、达人关注、限时秒杀及个性化配置推荐等功能。

- **优惠券秒杀：** 使用 Redis 计数器 + Lua 脚本实现库存原子扣减，结合分布式锁防止超卖，响应时间控制在 50ms 内
- **缓存设计：** 构建多级缓存体系，使用 Redis 缓存商品详情与评论，结合布隆过滤器解决缓存穿透问题
- **附近门店：** 基于 Redis GeoHash 实现“附近线下品牌门店”查询，提升本地化服务能力
- **用户签到：** 使用 Redis BitMap 统计每日签到状态，支持连续签到奖励与活跃度分析
- **UV统计：** 采用 Redis HyperLogLog 实现高精度去重统计，降低内存占用 90%+
- **项目部署：** 基于 Nginx 实现反向代理与负载均衡，结合 Docker 快速部署服务集群
